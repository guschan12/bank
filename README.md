# Bank

## Описание cron



```bash
Процесс начисления процентов сработает если перейти http://bank/calculate/percent
Процесс списания комиссии сработает если перейти http://bank/calculate/commission
По идеие cron начисления процентов отрабатывает каждый день, а по списанию комиссии каждое 1-е число на несколько часов позже первого cron`a
```
